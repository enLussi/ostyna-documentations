Gestion de votre base de données

Ostyna intègre un ORM pour vous permettre de mettre en place votre base de données simplement et rapidement. Avec de simples lignes de commandes, créez vos tables, ajoutez des valeurs, et récupérez les pour les utiliser dans vos controlleurs

Moteur de template intégré

Ostyna contient un moteur de templates simple et efficace qui vous permettra d’afficher des données en toute sécurité et de manière optimisée.
